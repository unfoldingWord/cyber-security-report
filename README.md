# Cybersecurity Standup Report

Automated daily security intelligence briefings from RSS feeds, powered by Claude.

This application fetches the latest security news, analyzes it with Claude's LLM, and generates structured reports with actionable intelligence—deployable as a scheduled Docker container or cron job.

## Quick Start

### Prerequisites
- Python 3.12+ or Docker
- Claude API access (`CLAUDE_CODE_OAUTH_TOKEN`)

### Installation

Using `uv`:
```bash
uv sync
```

Or using `pip`:
```bash
pip install -e .
```

### Run Once
```bash
python main.py
```

Reports appear in `reports/` as dated HTML and/or Markdown files.

## Key Features

- **Smart filtering**: Ignore noisy articles (e.g., job postings); prioritize critical topics (CVEs, ransomware)
- **Token-efficient**: Single LLM call; deterministic pre-processing (time window, deduplication, truncation)
- **Flexible output**: File-based reports + optional Zulip integration
- **Resilient**: Individual feed failures don't crash the pipeline
- **Branded**: HTML reports styled with unfoldingWord colors

## Configuration

Edit `config.yaml` to customize:
- RSS feeds
- Filter keywords (ignore/include lists)
- Report time window
- Output format (HTML, Markdown, or both)
- Zulip integration (optional)

See [IMPLEMENTATION.md](IMPLEMENTATION.md) for detailed configuration options.

## Usage

### Schedule with Cron
```bash
0 8 * * 1-5 cd /path/to/project && python main.py >> /tmp/cyber-sec-report.log 2>&1
```

### Run in Docker
```bash
docker build -t cyber-security-report .
docker run --rm \
  --env-file .env \
  -v $(pwd)/config.yaml:/app/config.yaml:ro \
  -v $(pwd)/reports:/app/reports \
  cyber-security-report
```
